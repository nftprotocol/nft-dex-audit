# NFT DEX

https://nft-dex.org

## Get started

Install:
```
nvm use 12
npm i
```

Configure:
```
# Edit secrets_example.json add mnemonic and infura key and save as secrets.json
```

Deploy contracts on ganache mainnet fork:
```
npx truffle migrate --network development --reset
```

Documentation:
```
open docs/html/index.html
```
